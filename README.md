## 実行方法について

### エントリーポイント
本プロジェクトのエントリーポイントとなるファイルは  
**`One_hour_dungeon_Next.py`** です。

データベース接続設定がありますが、**DB に接続しなくても動作可能**です。

---

## macOS 版の起動方法

macOS では **.app 形式のアプリ**を利用できます。  
配布版フォルダ内に **app ファイル**が含まれています。

1. `PythonGame` フォルダごとデスクトップへ移動してください  
2. フォルダ内の **app ファイル**を起動することでゲームが実行できます

---

## Windows 版の起動方法

Windows 版も配布しています。

起動には以下のファイル／フォルダが **MyGame.exe と同じディレクトリに配置**されている必要があります。

- `sound` フォルダ  
- `image` フォルダ  
- `Freesansbold.ttf`  

これらを準備した上で **MyGame.exe** を起動してください。
